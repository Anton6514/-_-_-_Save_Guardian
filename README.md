# Сохранятор сохранений или Save Guardian

Скрипт для автоматизации бэкапов сохранений с временным интервалом и определенным количеством слотов.

## Описание

Этот репозиторий содержит два скрипта на языке Windows Batch (`.bat`), предназначенные для создания периодических резервных копий папки с сохранениями игры во время её работы. Скрипт автоматически управляет количеством слотов резервных копий, удаляя самые старые, чтобы экономить место на диске.

## Файлы

*   `start_game_with_backup.bat`: Основной скрипт. Редактируйте пути внутри него, чтобы указать на исполняемый файл вашей игры, папку с сохранениями и желаемое место для бекапов.
*   `backup_loop.bat`: Вспомогательный скрипт, который выполняет копирование и управление слотами. Должен находиться в той же папке, что и основной скрипт.

## Установка и использование

1.  **Скачивание:** Скачайте файлы `start_game_with_backup.bat` и `backup_loop.bat` в одну папку на вашем компьютере.
2.  **Настройка:** Откройте `start_game_with_backup.bat` в любом текстовом редакторе (например, Блокнот).
    *   Найдите строку `set "GAME_EXE_PATH=..."` и измените путь справа на **полный путь к исполняемому файлу (.exe) вашей игры**.
    *   Найдите строку `set "GAME_SAVE_DIR=..."` и измените путь справа на **путь к папке, где ваша игра хранит файлы сохранений**. Обычно это папка вроде `AppData\LocalLow\Developer\GameName` или `Saved\SaveGames` внутри папки с игрой или в документах пользователя.
    *   Найдите строку `set "BACKUP_BASE_DIR=..."` и измените путь справа на **папку, куда вы хотите сохранять резервные копии**.
3.  **Настройка параметров резервного копирования:** Откройте `backup_loop.bat` в текстовом редакторе.
    *   Найдите строку `set "INTERVAL_SECONDS=3600"`. Число `3600` означает количество **секунд** между копиями (3600 секунд = 1 час). Измените это число на нужное вам (например, `1800` для 30 минут, `7200` для 2 часов).
    *   Найдите строку `set "MAX_SLOTS=12"`. Число `12` означает **максимальное количество** хранимых резервных копий (папок). Измените это число на желаемое количество слотов (например, `6`, `24`).
4.  **Запуск:** Запускайте `start_game_with_backup.bat` вместо того, чтобы запускать игру напрямую.
    *   Скрипт запустит фоновый процесс, который будет копировать папку сохранений с заданным интервалом, пока игра запущена.
    *   Когда вы закроете игру, процесс резервного копирования также автоматически остановится.
