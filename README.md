# Сохранятор сохранений или Save Guardian

Скрипт для автоматизации бэкапов сохранений с временным интервалом и определенным количеством слотов.

## Описание

Этот репозиторий содержит скрипты на языке PowerShell (`.ps1`), предназначенные для создания периодических резервных копий папки с сохранениями игры во время её работы. Скрипт автоматически управляет количеством слотов резервных копий, удаляя самые старые, чтобы экономить место на диске. Также включает создание начального бэкапа при запуске игры.

## Состав проекта

*   `start_game_and_backup.ps1`: **Основной скрипт**. Запускает игру, фоновый скрипт бэкапа и немедленно выполняет начальный бэкап. Отслеживает процесс игры и останавливает фоновый бэкап при её закрытии. **Параметры** (путь к игре, папке сохранений, папке бэкапа, интервал и количество слотов) **устанавливаются внутри файла**.
*   `backup_loop.ps1`: **Фоновый скрипт**. Выполняет копирование папки сохранений с заданным интервалом, пока процесс игры запущен. Автоматически управляет количеством слотов резервных копий, удаляя старые. Запускается из `start_game_and_backup.ps1`.
*   `initial_backup.ps1`: **Скрипт начального бэкапа**. Создаёт одну резервную копию папки сохранений в момент запуска. Запускается из `start_game_and_backup.ps1`.
*   `run_backup_manager.bat`: **Вспомогательный .bat файл**. Удобный способ запустить `start_game_and_backup.ps1` из Windows Explorer (двойным кликом). Запускает PowerShell с нужными параметрами.

## Требования

*   **Windows 7 или выше** (для работы PowerShell).
*   **PowerShell 5.1 или выше** (обычно встроен в современные версии Windows).
*   **Разрешение на выполнение скриптов PowerShell** (см. раздел "Установка").

## Установка и настройка

1.  **Скачивание:** Скачайте все файлы (`start_game_and_backup.ps1`, `backup_loop.ps1`, `initial_backup.ps1`, `run_backup_manager.bat`) в одну папку на вашем компьютере.
2.  **Настройка основного скрипта:** Откройте `start_game_and_backup.ps1` в любом текстовом редакторе (например, Блокнот).
    *   Найдите параметр `$GameExePath` и измените значение по умолчанию на **полный путь к исполняемому файлу (.exe) вашей игры**.
    *   Найдите параметр `$GameSaveDir` и измените значение по умолчанию на **путь к папке, где ваша игра хранит файлы сохранений**. Обычно это папка вроде `AppData\LocalLow\Developer\GameName` или `Saved\SaveGames` внутри папки с игрой или в документах пользователя.
    *   Найдите параметр `$BackupBaseDir` и измените значение по умолчанию на **папку, куда вы хотите сохранять резервные копии**.
    *   (Опционально) Найдите параметры `$IntervalSeconds` (по умолчанию `3600` секунд = 1 час) и `$MaxSlots` (по умолчанию `12`) и измените их на желаемые значения.
3.  **Настройка политик выполнения (если необходимо):**
    *   Если при запуске появляется ошибка вроде `File ... cannot be loaded because running scripts is disabled on this system.`, нужно изменить политику выполнения скриптов для текущего пользователя.
    *   Откройте **PowerShell** от имени **Администратора**.
    *   Выполните команду: `Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser`
    *   Подтвердите изменение (нажмите `Y`).
    *   Закройте окно PowerShell администратора.
4.  **Запуск:** Дважды кликните по файлу `run_backup_manager.bat`. Это запустит основной скрипт `start_game_and_backup.ps1` через PowerShell.

## Использование

*   Запускайте `run_backup_manager.bat` вместо того, чтобы запускать игру напрямую.
*   Скрипт запустит игру и фоновый процесс, который будет копировать папку сохранений с заданным интервалом, пока игра запущена.
*   Будет создан **начальный бэкап** сразу после запуска игры.
*   Когда вы закроете игру, процесс резервного копирования также автоматически остановится.
*   Резервные копии хранятся в папке, указанной в `$BackupBaseDir`, в подпапках с именами вида `Save_Backup_YYYYMMDD_HHMMSS`.
